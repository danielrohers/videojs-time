# videojs-time

Get the time watched video

## Installation
[Node.js](https://nodejs.org)
```bash
[sudo] npm install
```

## Starting

Compile files
```bash
gulp dist
```

Watch files
```bash
gulp watch
```

View example
```bash
npm start
```
